The Alura Plus project is a landing page that simulates an educational streaming platform, highlighting plans, benefits, and service features.

The development followed a front-end developer workflow, transforming a Figma layout into a functional web application, with attention to visual fidelity, responsiveness, and HTML/CSS best practices.

## Technologies Used
- HTML5;
- CSS3 with Flexbox and Grid.

## Features
- Semantic HTML page structure;
- Layout organization using Flexbox and Grid;
- Well-structured sections for content presentation;
- Buttons and call-to-action elements (CTA);
- Styling based on the Figma design.

## Design
The project layout was provided by Alura via Figma, and the implementation aimed to maintain fidelity to the original design, including:

- Typography;
- Spacing;
- Colors;
- Visual components.
